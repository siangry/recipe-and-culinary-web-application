# Recipe and Culinary Web Application
(Server Side Web Application)

An interactive recipe and culinary web application.
There are 4 modules covered in this web application:
1. Recipe Management: To create, store and manage recipes
2. Meal Planning: To plan meals by selecting recipes or adding custom meal entries
3. Community Engagement: To interact through discussions, comments, and ratings
4. Cooking Competition: To participate in recipe contest by submitting their recipes to engage in friendly competition

## Tech Stack
- PHP
- HTML
- CSS
- JavaScript
- Bootstrap
- Relational database
